# Rock Paper Scissors - Pixel Arcade

8-bit 레트로 아케이드 컨셉의 React 가위바위보 게임

## 🎮 Live Demo
[배포 링크]

## 🛠 Tech Stack
- React 18 (Hooks: useState, useRef, useEffect)
- Vite
- CSS3 (Custom Properties, Keyframes, Flexbox)

## ✨ Features
- 룰렛 연출 (setInterval + setTimeout 비동기 처리)
- 동적 className으로 결과별 시각 피드백
- 점수판 (W/D/L 카운트)
- 8비트 레트로 디자인 (CRT 스캔라인, 픽셀 폰트)
- 반응형 (모바일/태블릿/데스크탑)
- 메모리 누수 방지 (useEffect cleanup)

## 🎨 Design Concept
- Press Start 2P + VT323 픽셀 폰트
- 네온 컬러 팔레트 (Yellow/Pink/Cyan/Green)
- CRT 모니터 스캔라인 효과
- 픽셀 그림자 + 호버 인터랙션

## 📚 Learning Points
- React 컴포넌트 재사용성 (단일 Box로 Computer/You 표현)
- 불변성 유지 (스프레드 연산자 + 함수형 업데이트)
- 동적 className 패턴 (Template Literal)
- 비동기 처리 (인터벌/타임아웃 + ref 보관)
- CSS 변수 기반 디자인 시스템
